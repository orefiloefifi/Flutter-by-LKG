# 🌸 Flutter By LKG

**Flutter By LKG** is a modern luxury beauty and feminine care website built using HTML, CSS, and JavaScript. The website showcases premium feminine hygiene and beauty products with an elegant, responsive design focused on empowering women through confidence, comfort, and sophistication.

---

## ✨ Features

- 🌸 Beautiful luxury landing page
- 💜 Responsive mobile-first design
- 🛍️ Product showcase section
- 📸 Interactive image gallery with lightbox
- 📖 About the brand section
- 📱 Smooth scrolling navigation
- ✨ Hero typing animation
- 🎯 Animated product hover effects
- ⬆️ Back-to-top button
- 📲 Social media links
- 📧 Contact section
- ⚡ Lightweight with no external frameworks

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Poppins)

### Clone the repository

```bash

```

### Navigate into the project

```bash

```

### Open the website

Simply open:


- VS Code Live Server
- GitHub Pages
