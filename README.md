# 📱 Overview

**Flutter by LGK** is a modern developer portfolio created to showcase Flutter applications, UI/UX projects, and cross-platform mobile development expertise. Inspired by premium digital experiences, the website combines elegant design, smooth animations, and responsive layouts to present projects in a visually engaging way.

The platform highlights technical skills, featured work, development services, and creative solutions while demonstrating modern frontend development practices.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/orefiloefifi/Flutter-by-LGK.git
```

## Navigate into the Project

```bash
cd Flutter-by-LGK
```

## Run the Website

Open `index.html` in your browser or launch the project using **Live Server** in Visual Studio Code.
